# Security Review

## Scope

Welche Dateien, Workflows oder Services sind betroffen?

## Secrets

- [ ] Keine Secrets in tracked Files.
- [ ] Neue Secrets sind als GitHub Secrets dokumentiert.
- [ ] Nicht benoetigte Secrets wurden nicht kopiert.

## Datenschutz

- [ ] Keine PII in Repo, Logs, Outputs, Issues oder Chat.
- [ ] Outputs enthalten nur freigegebene Markdown-Artefakte.

## Runtime

- [ ] Deploy-Ziel ist explizit.
- [ ] Rollback ist dokumentiert.
- [ ] Smoke-Check ist benannt.
