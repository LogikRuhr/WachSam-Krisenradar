# WachSam

**Deutschland-zentriertes Krisen- und Haushalts-Auswirkungsradar.**

WachSam übersetzt Krisen, Kaskaden und Systemstress in verständliche Auswirkungen auf Kosten, Versorgung und Stabilität deutscher Haushalte.

## Aktiver Produktcode

Der aktive Produktcode liegt unter `v02/`.

- `v02/web/` — Next.js 15 App Router, TypeScript, Public UI, Auth, Admin/UI
- `v02/db/` — Drizzle-Schema, Migrations, Seed-Daten
- `v02/intelligence/` — Python-Adapter, Source Health, Evidence-first Intelligence
- `v02/infra/` — Docker-/Runtime-Artefakte

Live: `https://wachsam.ruhrlogik.de`
